# Image Citation

Images in this directory are from:

> **Digital Image Processing, 3rd Edition**
> Rafael C. Gonzalez and Richard E. Woods
> Pearson, 2008
> ISBN: 978-0131687288

The authors provide these images freely for educational use at:
https://www.imageprocessingplace.com/DIP-3E/dip3e_book_images_downloads.htm

Only the subset of images used in the tutorials is included in this repository.
