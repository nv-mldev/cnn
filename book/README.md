## How this book relates to the code

```
book/          ← narrative: motivation, derivations, intuition (you are here)
book/figures/  ← all simulation figures (generated by tutorials/)
tutorials/     ← runnable Python: one script per section, produces figures
```

Run a tutorial script to regenerate figures:
```bash
uv run python tutorials/00_introduction_to_digital_images/part1_digitisation.py
```
